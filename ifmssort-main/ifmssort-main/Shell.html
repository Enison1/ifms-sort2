ShellSort

Shell sort é principalmente uma variação de Insertion Sort . Na classificação por inserção, movemos os elementos apenas uma posição à frente. Quando um elemento precisa ser movido muito à frente, muitos movimentos estão envolvidos. A ideia do ShellSort é permitir a troca de itens distantes. No Shell sort, fazemos o array classificado por h para um grande valor de h. Continuamos reduzindo o valor de h até que se torne 1. Diz-se que um array é classificado por h se todas as sublistas de cada h'ésimo elemento forem classificadas.

Complexidade

Complexidade: O(n2)
